Förklaring av begrepp

Element:

Ett element består av en start och slut tag. Html är ett element och body är ett annat element tex. Här nere är det en bild på "element".

HTML:

HTML använder man för att skapa webbsidor och det står för Hyper Text Markup Language. HTML är ett viktigt element och hela sidan bygger på det om man gör en webbsida i html.

ID:

Man kan använda ID när man vill identifiera element på en sida. Det ID man använder får man bara ha en gång i dokumentet. Här nere är en bld på flera likadana IDs som då också är ogiltiga.

Class:

Class kan man använda när man vill "koppla" något till sitt style sheet, tex kan man skriva class="container" i en div tag och komma åt den i tex style.css.

Meta:

Meta är en tag och kolla på bilden för att se hur den ser ut. Det den gör är att definiera metadata vilket helt enkelt är information om data. Det är webbläsare som använder metadatan.

Responsive:

Responsive web design gör så att webbsidan ser bra ut på alla enheter som mobil, surfplatta och dator. Man använder HTML och CSS för att webbsidan automatiskt ska anpassa sig efter vilken enhet den är på. Tex ändra storleken på texter.

http:

Står för Hypertext Transfer Protocol och det är ett kommunikationsprotokoll som används för att transfera eller överföra hemsidors data. Http är protokollet som WWW (world wide web) använder sig av för att se hur meddelanden har definierats och vilka åtgärder webbservrar ska vidta vid olika kommandon.

https:

Https är är stort sagt en säkrare version av http. S:et står för secure. Skillnaden mellan http och https är att i http-protokollet krypterar informationen som skickas mellan klient och server med TLS eller SSL.

compressed data:

Datakompression är när när datan omkodas så att färre informationsbärande enheter behöver användas. Med andra ord så tar det upp mindre utrymme på datorn.

CDN:

CDN står för content delivery network och är ett innehållsleveransnätverk. Meningen med nätverket är att tillandahålla nättjänster nära användarn med syfte att öka driftsäkerheten och prestandan.

local bootstrap:

Bootstrap är en viss CSS. Det är en massa kod som hjälper din webbsida att anpassa sig till vilken enhet den är på. På tex en dator kan den se ut på ett visst sätt och på mobilen ser den annorlunda ut eftersom den är en annan storlek på skärmen.

JQuery:

JQuery är helt enkelt ett Javascript-bibliotek och meningen med det var att förenkla HTML; DOM och CSS-modifikation. Skaparen av bibliotket är John Resig och han släppte det 2006.

Flexbox:

Flexbox är en viss layout i CSS. Man kan använda flexbox-layouten för att få elementen man har i en container att ordnas automatiskt beroende på vilken skärmstorlek den vissas på. Det finns flera olika "layouts" i flexbox. Här nere är en bild på olika layouts.

Grid:

Grid eller CSS Grid är en annan viss layout som är som ett rutnät där ett antal linjer korsar ytan såväl horisontelt som vertikalt. Så här ser grid ut.

Code snippets:

Det är ett x antal färdigskriven rader av kod som man kan kopiera och återanvända. Oftast är dessa stycken av kod formellt definierade operativa enheter som ska införlivas i större programmeringsmoduler.
